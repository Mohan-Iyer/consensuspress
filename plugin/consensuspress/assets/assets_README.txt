ConsensusPress by Seekrates — WordPress.org Assets
===================================================

Place the following image assets in this directory before submitting to WordPress.org.
All images must be PNG format.

REQUIRED ASSETS
---------------

banner-772x250.png
  Plugin directory banner (standard resolution)
  Dimensions: 772 x 250 px
  Used on: Plugin directory listing page header

banner-1544x500.png
  Plugin directory banner (retina / high-DPI)
  Dimensions: 1544 x 500 px
  Used on: High-DPI displays in plugin directory

icon-128x128.png
  Plugin icon (standard resolution)
  Dimensions: 128 x 128 px
  Used on: Plugin search results and listings

icon-256x256.png
  Plugin icon (retina / high-DPI)
  Dimensions: 256 x 256 px
  Used on: High-DPI displays in plugin directory

RECOMMENDED SCREENSHOTS
------------------------

screenshot-1.png  — Create Mode admin page (topic input + progress)
screenshot-2.png  — Rescued post diff view (before/after comparison)
screenshot-3.png  — Consensus meta box on post edit screen (score, champion, risk)

Screenshots are referenced in readme.txt == Screenshots == section in order.
File names must match exactly: screenshot-1.png, screenshot-2.png, etc.

SUBMISSION NOTES
----------------

WordPress.org requires assets to be committed to the /assets/ directory of the
SVN repository, NOT to the /trunk/ directory. The deploy_to_wp_org.sh script
handles this separation automatically.

See: https://developer.wordpress.org/plugins/wordpress-org/plugin-assets/
